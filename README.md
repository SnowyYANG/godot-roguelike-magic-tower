# Godot RoglikeMagicTower Example

A simple rogue-like magic tower game developed using Godot.

## Features
* Map: obstacles, monsters, items, stairs
* Player: move, fight
* Simple animations

## About
* Program: Snowy
* Art: Way, vurmux (Urizen 1Bit Tileset)
* Music: Abstraction
